Decent Dialogue Chatbot

Role

UX Research, UX Design

Tools

Adobe Illustrator, Sketch, Python

Team

Jingfei Lin (Programmer)

Overview

Decent Dialogue is a chatbot simulating conversations among celebrities.

No matter whether the celebrity is still alive, no matter what language the celebrity speaks, as long as some original speeches or writings can be found and translated, we can create a chatbot based on any specific styles and characteristics. By simply selecting two or more celerities, our user can create a decent dialogue among them.

This chatbot is built by the language Python, deconstructing and analyzing different styles of dialogues, in order to create a playful and interactive experience.
