Amino App Store

Role

UX Researcher, UI/UX Designer

Methods

Mobile Audit, Contextual Inquiry, User Journey Map, Competitive Analysis, Wireframe

Tools

Sketch, Adobe Illustrator, Adobe Photoshop, Adobe Premiere

Overview

Amino Apps is a mobile-first social networking platform developed by Narvii. The goal of the social network is to provide communities for fanbases and topics.

In order to improve the UI of the main Amino store homepage and sticker pack detail view, and provide users a better experience of purchasing digital goods, I conducted this design for the Amino App Store Interface.
