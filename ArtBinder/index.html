ArtBinder iOS App

Role

UX Researcher, UX/UI Designer

Methods

Contextual Inquiry, User Journey Map, Wireframe, Prototype, User Test, Iteration

Tools

Balsamiq Mockup, Sketch, Miro

Overview

ArtBinder is a startup company that provides a Content Management System tool for Art Professionals.

Currently ArtBinder clients are mainly using the web CMS, and the following is an app critique of the current ArtBinder iOS app, as well as the user research and UX/UI re-design of the app.

Side Project: ArtBinder Brochure Request

Role

UX/UI Designer

Methods

Contextual Inquiry, Customer Journey Map, Competitive Analysis, Wireframe

Tools

Sketch, Principle, Adobe Illustrator, Adobe Photoshop, Adobe Premiere, Realtime Board

Overview

This is a 3 step flow for users on mobile devices to visit ArtBinder's marketing website and request a brochure.

ArtBinder wants to give potential customers (art professionals) the ability to request a brochure from ArtBinder.com. After requesting, a physical brochure will be mailed, with the general information about the features and pricing of ArtBinder's Basic and Premium products.
